Where Has This Gotten Us

Toolbelt

cli.go

  app := cli.NewApp()
  app.Name = "boom"
  app.Usage = "make an explosive entrance"
  app.Action = func(c *cli.Context) {
    println("boom! I say!")
  }

Cobra

  • Global and nest flags
  • Easy app generation (indluding help)
  • 12-factor compatible
  • Posix flags

Viper

  • Find, load, and unmarshal a configuration file in JSON, YAML, HCL, etc.
  • Allow you to set default values for your different config options.
  • Allow you to set override values command line flags.

Godep

  • Pull in external packages to reduce outside dependencies
  • Cut down on network i/o by keeping code self-contained

Logrus

  • Fully structured logging
  • Api compatible with std go logging
  • Ascii or json out
  • Multiple logging instances

Goconvey

  • Integrated with go test
  • Pretty web ui (if care about such things)
  • Coverage reports

Geting From Here To There

Writing the code

Any editor as long as its vim

Cobra provides a simple scaffolding command

Viper provides a simple way to deal with configuration

Godep provides a simple and clean vendoring solution

Goconvey provides a simple testing and coverage framework

Logrus provides a simple logging framework

Vagrant/Docker provides a simple local test environment

Building the code

Commit to Github

Building is managed via jenkins through a Makefile

Compiling is done within a container of the dev's choice

Binaries are versioned and pushed to Artifactory and Github

Libraries are marked as building

Deploying the code

Via Curl from Artifactory in Dockerfiles

Via Chef ark cookbook on vm's
